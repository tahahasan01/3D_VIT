# SMPL body models

Place your SMPL model files here after registering at https://smpl.is.tue.mpg.de/

Required files (rename from the download if needed):

- `SMPL_MALE.pkl` – male model (e.g. from `basicModel_m_lbs_10_207_0_v1.1.0.pkl`)
- `SMPL_FEMALE.pkl` – female model (e.g. from `basicModel_f_lbs_10_207_0_v1.1.0.pkl`)

The app uses these when "Use SMPL body" is selected in the body measurements step.
